# Brand Guide — The Candlelit Kitchen

Aivyx's design language: warm, inviting, and focused.

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Candle Amber | `#F4A261` | Primary accent, CTAs |
| Sage Green | `#7FB069` | Success states |
| Warm Charcoal | `#1A1A2E` | Background (dark) |
| Soft Cream | `#FAF3E8` | Background (light) |
| Smoke | `#2D2D44` | Card surfaces |
| Error | `#E76F51` | Errors, danger |
| Warning | `#E9C46A` | Warnings |

## Typography

| Weight | Font | Usage |
|--------|------|-------|
| Headings | **Outfit** (600–700) | Titles, headers |
| Body | **Outfit** (400) | Paragraph text |
| Code | **JetBrains Mono** | Code blocks |

## Guidelines

- Dark mode default
- Rounded corners (8–12px)
- Subtle hover animations
- Gradient accents: warm amber to sage green
